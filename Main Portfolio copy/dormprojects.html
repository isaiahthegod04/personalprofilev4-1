Dormbell

I love having guests over, to make sure these guests don't catch me by surprise, I built a "dormbell" that plays clips from randomly selected songs when pressed!

How it works

Skills

Improvements

  • Arduino IDE code
    I used an Arduino UNO and added code that output the correct pitches in the correct order
  • Music selection
    This code randomly selected one of five manually uploaded songs and played them through a buzzer
  • Pitches
    To ensure each song was correct, I created a function that held the correct pitches and then recalled the function when needed
  • Arduino IDE
    This project used an Arduino UNO and C++

    To prevent any damage to the Arduino, I had to ensure my circut was not under too much voltage which took trial and error

    If I were to remake this project, I would try to simplify the code as I believe there is many parts that can be simplified
  • Output
    I plan to rebuild this project, and instead replace the piazza buzzer with stepper motors to create a better pitch output

Door Opener

When I first moved into my dorm, it took some time to get used to the automatic locking mechanism on my door and I would frequently lock myself out, so, I built a keypad-activated unlocking system for the next time I locked myself out

How it works

Skills

Improvements

  • Arduino IDE code
    I used an Arduino UNO and added code that would start a motor when it detected the correct keypad input
  • Motor
    I modeled a two-way gear train that would push up the door handle from the inside, allowing me to open the door
  • Arduino IDE
    This project used an Arduino UNO and C++
  • CAD
    I used cad to create a physical device to open the door
  • Circutry
    I had to wire the keypad, motors and opener and connect everything from inside and outside my door
  • CAD
    If I were to remake this project, I would try to remodel the door mechanism, as it is annoying to move out of the way when normally opening the door
